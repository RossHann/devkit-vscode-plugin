Viewing Snapshot Information

Prerequisites

A profiling task is being performed.

Procedure

  1. On the home page of the Java Profiler WebUI, click the target task name in the Profiling Records area.

    The Overview tab page is displayed.

  2. On a page that supports the snapshot function, click in the upper right corner to save the snapshot of the current page. The snapshot is named after the time when the snapshot is saved.

    A maximum of five snapshots can be saved for each of the following tab pages:
    • Heap Dump
    • File IO and Socket IO under IO
    • JDBC, JDBC Connection Pool, MongoDB, Cassandra, and HBase under Database
    • HTTP Requests

  3. Click the Snapshot tab.

    Information of saved snapshots is displayed, as shown in Figure 1. For details about the parameters, see the corresponding sections on each page.

    When Internet Explorer is used, the snapshot comparison table does not support customized table heading.

    Figure 1 Snapshot tab page

    Select a snapshot and click in the left pane to delete the snapshot.

    Click in the left pane and select snapshots to compare.
