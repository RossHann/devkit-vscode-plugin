Querying Hotspot Information

Prerequisites

A profiling task is being performed.

Hotspot analysis indicates that the perf profiler is used to collect CPU stack top information at some time points and collect statistics on the hotspot methods in the current JVM. Queries are performed in the form of an inverted flame graph.

Querying Hotspot Information

  1. On the home page of the Java Profiler WebUI, click the target task name in the Profiling Records area.

    The Overview tab page is displayed.

  2. Click the Hotspot tab.

    On the Hotspot tab page, click Create Hotspot Analysis to view the hotspot information, as shown in Figure 2. Table 1 describes the parameters.

    • If the server does not support hotspot analysis, the Create Hotspot Analysis button is dimmed. Tips about the reason of unavailability and method for using this function are given.
    • During startup, you can filter function methods that do not need analysis. If you specify methods that do not require analysis, you can click View Details to get more information.
    • Full-screen display is supported. You can click the corresponding graph to view the method details.
    • You can click Recreate Hotspot Analysis to re-create a hotspot analysis task.

    Table 1 Parameters for creating a hotspot analysis task

    Parameter

    Description

    Sampling Time (s)

    Data sampling time.

    Sampling Interval (ms)

    Interval between data samplings.

    Sampling Event Type

    Type of the sampled data.

    Maximum Traced Stack Depth

    The maximum stack depth of the sampled data.

    Method Without Analysis

    User specified data that does not need to be sampled.

    Method With Analysis

    Data that must be sampled.

    Trigger Analysis By

    Method that triggers the analysis.

    End Analysis By

    Method that triggers the ending of the analysis.

    Kernel Mode Analysis

    Kernel mode invoking status.

    Clock Cycle (Percentage)

    Percentage of the clock cycle.

    Address

    Memory address of a process.

    Assembly

    Assembly instruction.
