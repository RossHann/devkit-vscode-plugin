Running the Installation or Uninstallation Script on the CLI

Installation

The command for running the Kunpeng Hyper Tuner installation script is in the following format. Table 1 lists the related parameters.

./install.sh [-i | -r | -u] [-a | -j | -s | -c | -l] [-d=install path] -ip=web server IP [-p=web server port] [-mip=mallumad IP] [-mp=mallumad port] [-jip=java-perf IP] [-jp=java-perf port] -jh=java home

  • If only the System Profiler is installed, the command does not contain -jip, -jp, and -jh.
  • If only the Java Profiler is installed, the command does not contain -mip and -mp.
  • Conventions for the command line formats:

    Boldface: The keywords that must keep unchanged in a command line are in boldface.

    Italic: Command arguments that must be replaced by specific values are in italic.

    [ ]: Items in square brackets [ ] are optional.

    [ x | y | ... ]: Optional items are grouped in brackets and separated by vertical bars. One item is selected or no item is selected.

Table 1 Installation command parameters

Command

Parameter

Description

-i/--install

-r/--rollback

-u/--upgrade

-l

-c

-

Operation type. This parameter is optional.

Default value: -i/--install

The options are as follows:

  • -i/--install: installs the tool.
  • -r/--rollback: rolls back the tool.
  • -u/--upgrade: upgrades the tool.
  • -l: checks dependency items before the installation.
  • -c: clears previously installed tools.

-a/--all

-j/--java

-s/--system

-

Tool to be installed. This parameter is optional.

Default value: -a/--all

The options are as follows:

  • -a/--all: installs the System Profiler and Java Profiler.
  • -j/--java: installs only the Java Profiler.
  • -s//--system: installs only the System Profiler.

-d/--directory

install path

Installation path. This parameter is optional.

Default value: /opt

-ip/--ip

web server IP

IP address of the server on which the tool is installed. This parameter is mandatory.

-p/--port

web server port

HTTPS port of the Kunpeng Hyper Tuner. This parameter is optional.

Default value: 8086

Value range: 1024 to 65535

-mip/--mip

mallumad IP

Internal IP address of the System Profiler. This parameter is optional.

The default value is the web server IP address specified by -ip/--ip.

-mp/--mport

mallumad port

Port used for communication between the web server (server running the System Profiler) and the agent node server. This parameter is optional.

Default value: 50051

Value range: 1024 to 65535

-jip/--jip

java-perf IP

Internal IP address of the Java Profiler. This parameter is optional when the Java Profiler needs to be installed.

The default value is the web server IP address specified by -ip/--ip.

-jp/--jport

java-perf port

Internal communication port of the Java Profiler. This parameter is optional when the Java Profiler needs to be installed.

Default value: 9090

Value range: 1024 to 65535

-jh/--java_home

java home

JDK (Open JDK 11) installation path in the server. This parameter is optional for installing the Java Profiler. You can run the echo $JAVA_HOME command to obtain the path.
NOTE:
  • When configuring the IP address of the tool, enter the local server IP address (Run ip addr on Linux to query the IP address).
  • If -ip/--ip, -mip/--mip, and -jip/--jip are set to the same IP address, -p/--port, -mp/--mport, and -jp/--jport must be different ports.
  • If the configured port conflicts with a port in use, the system automatically increases the port number by 1. If no available port is found after 100 cycles, the installation will be terminated.

Example: To install the System Profiler and Java Profiler, run the following command:

./install.sh -i -a -d=/opt -ip=10.247.81.189 -p=8086 -mip=10.247.81.189 -mp=50051 -jip=10.247.81.189 -jp=9090 -jh=/opt/jdk11

Or:

./install.sh -ip=10.247.81.189 -jh=/opt/jdk11

Uninstallation

The command for running the Kunpeng Hyper Tuner uninstallation script is in the following format. Table 2 describes the parameters.

./hyper_tuner_uninstall.sh [-a | -j | -s]

Conventions for the command line formats:

  • Boldface: The keywords that must keep unchanged in a command line are in boldface.
  • [ x | y | ... ]: Optional items are grouped in brackets and separated by vertical bars. One item is selected or no item is selected.
Table 2 Uninstallation command parameters

Command

Parameter

Description

-a/--all

-j/--java

-s/--system

-

Tool to be uninstalled. This parameter is optional.

Default value: -a/--all

The options are as follows:

  • -a/--all: uninstalls the System Profiler and Java Profiler.
  • -j/--java: uninstalls the Java Profiler.
  • -s/--system: uninstalls the System Profiler.

Example: To uninstall the System Profiler and Java Profiler, run the following command:

./hyper_tuner_uninstall.sh -a

Or:

./hyper_tuner_uninstall.sh
