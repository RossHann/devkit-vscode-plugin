Disabling SELinux

  • To disable SELinux temporarily, perform the following steps:
    1. Run the sestatus command to query the SELinux status.

      If Current mode is enforcing, the SELinux enforcing mode is enabled.

    2. Set the SELinux mode to permissive:

      setenforce 0

    3. Run the sestatus command to query the SELinux status.

      If Current mode is permissive, the SELinux enforcing mode is disabled temporarily.

    The settings become invalid after the server is restarted. When the tool is used, service exceptions may occur.

  • To disable SELinux permanently, perform the following steps:
    1. Run the sestatus command to query the SELinux status.

      If SELinux status is enabled, SELinux is enabled.

    2. Open the /etc/selinux/config file:

      vi /etc/selinux/config

    3. Press i to enter the edit mode and change SELINUX=enforcing to SELINUX=disabled.
    4. Press Esc, type :wq, and press Enter to save the settings and exit.
    5. Run the reboot command to restart the server.
    6. Run the sestatus command to query the SELinux status.

      If SELinux status is disabled, SELinux is disabled permanently.
