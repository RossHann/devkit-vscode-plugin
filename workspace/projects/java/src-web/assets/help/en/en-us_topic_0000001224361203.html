Deleting a Guardian

Prerequisites

  • A Guardian is added.
  • The SSHD service has been enabled on the server where the Guardian is installed.

Procedure

  1. Log in to the Java Profiler WebUI, locate the Guardian to be deleted, and click .
  2. Click OK in the dialog box displayed.

    • If the Guardian is offline, confirm the deletion by entering the user name and password and then click OK. If you click OK without entering the password, manually delete the executable file of the Guardian on the remote server.
    • If an error message indicating that the analysis assistant software cannot be installed is displayed when you delete the Guardian, check whether the password is correct and whether the current user is locked.
