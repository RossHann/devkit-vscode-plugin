Viewing Thread Dump Analysis Results

Prerequisites

A sampling analysis task is created and complete.

Procedure

  1. On the home page of the Java Profiler WebUI, click the target task name in the Sampling Analysis Records area.

    Click the CPU tab.

  2. Click the Thread Dump tab.

    On the Lock Analysis Diagram page, click the specified thread dump to view the thread lock analysis diagram, as shown in Lock Analysis Diagram.
    • In the lock analysis graph, threads are displayed on the left, and lock instances are displayed on the right. The color indicates the thread status. Green indicates Runnable, red indicates Blocked, yellow indicates Waiting, and orange indicates Timed_waiting.
    • The lock analysis diagram uses lines to show the lock occupation by threads. A solid line indicates that the lock is used by the thread, and a dashed line indicates that the thread is blocked on the lock instance and waits for other threads to release the lock.
    • Observation: If it is enabled, the threads and their locks or the locks and their requested threads will be highlighted. You can also click a lock and the lock and its requesting threads are highlighted.
    • Comparison: After enabling this mode, select the thread dump to be compared. By default, the comparison results of the threads and their locks are displayed. By using the comparison mode and the observation mode together, you can better observe the status change of threads and locks at two time points. Under the Comparison mode, you can enable Observation and click a lock to observe the status change of the lock and its requesting threads at two time points. The comparison mode can be enabled only after two or more thread dumps.
    • Deadlocks can be detected in the target Java application. You can locate the cause of a deadlock based on the thread dump.
    • You can click icons in the lower right corner to zoom in or zoom out the lock analysis diagram and view the diagram in full screen.
    Figure 1 Lock Analysis Diagram

  3. Select Raw data from the Lock Analysis Diagram drop-down list.

    On the page that is displayed, expand the target thread dump and click a thread name to view call stack information of the thread, as shown in Raw data.
