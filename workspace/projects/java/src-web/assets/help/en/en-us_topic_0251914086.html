Managing Profiling Tasks

Prerequisites

A profiling task is being performed.

Managing Profiling Tasks

On the Java Profiler WebUI, view the Profiling Records area, as shown in Figure 1.
Figure 1 Profiling Records area

Click next to Profiling Records to select and import a sampling analysis record file.

Importing a profiling task will stop the ongoing profiling process. Exercise caution when performing this operation.

Click next to a profiling task to export the online analysis record file and save it to the local PC.

Click next to a profiling task to stop the task.
