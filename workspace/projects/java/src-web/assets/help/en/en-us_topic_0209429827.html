Viewing Overview Information

Prerequisites

A profiling task is being performed.

Procedure

  1. On the home page of the Java Profiler WebUI, click the target task name in the Profiling Records area.

    The Overview tab page is displayed by default, as shown in Figure 1 and Figure 2. For details about the parameters, see Table 1.
    • The Parameters area lists the system parameters of the Java process. You can move the cursor onto a parameter to view details.
    • You can click to view the tuning suggestions on the current task.
    • You can click the Perform Thread Dump button to dump the current thread status for analysis and display on subsequent tab pages.
    • Click to export the analysis report on all tab pages (excluding the Memory, Hot Spot, GC Log, and Thread Dump tab pages), as shown in Figure 3.

    Table 1 Parameters on the Overview tab page

    Parameter

    Description

    Heap Memory/Non-heap Memory

    Used heap memory/Used non-heap memory

    Heap memory or non-heap memory used by the target Java application.

    Committed heap memory/Committed non-heap memory

    Heap memory or non-heap memory reserved for the target Java virtual machine (JVM).

    Physical Memory

    Memory Used by Java Processes

    Physical memory used by the target Java application.

    Free System Memory

    System memory reserved for the target JVM.

    GC Activity

    Pause Time

    Time when an application is paused due to GC.

    Class

    Loaded Classes

    Number of Java classes loaded.

    Thread

    Running Threads

    Number of threads in the running state.

    Waiting Threads

    Number of threads in the waiting state.

    Blocked Threads

    Number of threads in the blocked state.

    CPU Load

    System CPU Usage

    CPU usage of JVM processes.

    CPU Usage of Java Processes

    CPU usage of Java processes.

    Environment

    PID

    Process ID.

    Host

    IP address of the host.

    Main Class

    Name of the entry class of the program.

    Arguments

    Command line arguments.

    JVM

    Version information of the JVM.

    Java

    Version information of the JDK.

    Keyword

    Names of environment variables.

    Value

    Values of environment variables.
