Restarting the Kunpeng Hyper Tuner

  1. Use an SSH remote login tool to log in to the Linux CLI.
  2. Run the following commands to restart the Kunpeng Hyper Tuner. (/opt/hyper_tuner is the tool installation directory. Replace it with the actual directory.)

    cd /opt/hyper_tuner

    ./service.sh restart

    If the Kunpeng Hyper Tuner is installed on a Docker container, run the following commands:

    cd /opt/hyper_tuner

    ./service.sh restart -t docker
