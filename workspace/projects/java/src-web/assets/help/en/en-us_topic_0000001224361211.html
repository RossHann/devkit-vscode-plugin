Viewing GC Information

Prerequisites

A profiling task is being performed.

Procedure

  1. On the home page of the Java Profiler WebUI, click the target task name in the Profiling Records area.

    The Overview tab page is displayed.

  2. Click the GC tab.

    View the GC event information. The GC analysis page is displayed by default, as shown in Figure 1. Table 1 describes the parameters.
    • You can select GC Cause or Garbage Collector from the drop-down list and enter a keyword to quickly filter data.
    • You can click to view the tuning suggestions on the current task.
    • You can click to export the current GC report.
    Figure 1 GC Analysis
    Table 1 Parameters on the GC analysis tab page

    Parameter

    Description

    Timestamp

    Time when a GC event occurred.

    GC Cause

    Cause of GC.

    Garbage Collector

    Name of the garbage collector that performs GC.

    Memory Applied for GC

    Memory capacity applied for GC.

    Memory Before GC

    Memory capacity before GC.

    Memory After GC

    Memory capacity after GC.

    GC Memory Size

    Memory reclaimed by GC.

    GC Threads

    Number of threads used during GC.

    Pause Time (ms)

    Time when an application is paused due to GC.

    Collected GC Event Info

    GC Activities

    Displays the ratio of the GC pause time to the unit time (1 second) in an area chart.

    Memory

    Displays the memory for which the JVM applies, memory in use, and free memory in a stacked area chart.

    GC

    Displays the memory reclaimed by GC in an area chart.

    Pause Time (ms)

    Displays the application pause time caused by GC in an area chart.

    GC Threads

    Displays the number of threads used during GC in an area chart.

  3. Click GC Logs.

    Click Collect GC Log to view the analysis of key GC metrics, as shown in Analysis of key GC metrics. Parameters on the key GC metrics describes the parameters.

    • During GC log collection, the collection progress and the Cancel Collection button are displayed. You can click Cancel Collection to cancel the collection event.
    • When you move the mouse pointer to the collection diagram, the detailed information for the corresponding time is displayed.
    • If GC log data is not obtained, enable the needed log level. For details, see "Common Operations > Enabling a Log Level to Obtain GC Logs".
    • Click to save the current GC log information. The same GC log information can be saved only once. You can view and download the GC log information in the GC log list on the main page.

    Click Display to change the analysis type. Select GC Causes to view the GC cause analysis data, as shown in GC Causes.
