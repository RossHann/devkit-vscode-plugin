Operation Log Management

Prerequisites

You have logged in to the Java Profiler WebUI.

Only the administrator tunadmin can view and download the operation logs of all users. Common users can view and download only the operation logs of the current login user.

Procedure

  1. Click in the upper right corner and choose Logs.

    The Operation Log page is displayed, as shown in Figure 1. For details about the parameters, see Table 1.
    • When you log in to the system as a common user, you can click Operation Logs in the upper right corner of the page to open the operation log page.
    • When the number of common operation log entries reaches 2000, the tool exports the entries to the operation_logs.csv file in the /opt/hyper_tuner/user_management/logs directory (/opt/hyper_tuner is the tool installation directory, which can be customized), and clears data on the Common Log tab page. When a new operation_logs.csv is generated, the old one will be overwritten automatically.
    • The operation logs of the Java Profiler are stored for 30 days by default. The tool automatically checks operation logs at 01:00 every day and deletes expired log entries.
    Figure 1 Operation Log tab page
    Table 1 Parameters on the Operation Log tab page

    Parameter

    Description

    User Name

    User who performed the operation.

    Event

    Type of the operation performed.

    Result

    Operation result.

    IP

    IP address of the host on which the operation was performed.

    Time

    Time when the operation was performed.

    Detail

    Details about the operation.

  2. Click Download Log to download the log file in .csv format.
