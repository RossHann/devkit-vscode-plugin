Viewing Overview Information

Prerequisites

A sampling analysis task is created and complete.

Procedure

  1. On the home page of the Java Profiler WebUI, click the target task name in the Sampling Analysis Records area.

    The Overview tab page is displayed by default, as shown in Figure 1 and Figure 2. For details about the parameters, see Table 1.

    • The CPU Usage area displays the line chart of the CPU usage of the current process.
    • You can click to view the tuning suggestions on the current task.
    Figure 1 Overview tab page
    Figure 2 Overview (system environment information and environment variables)
    Table 1 Parameters on the Overview tab page

    Parameter

    Description

    CPU Usage

    JVM User Mode

    CPU usage of the JVM in user mode.

    JVM System Mode

    CPU usage of the JVM in kernel mode.

    Total system usage

    Total CPU usage of the computer.

    Java System Property

    Key

    Java system property key.

    Value

    Java system property value.

    System Environment

    CPU

    CPU type.

    Cores

    Number of CPU cores.

    Memory

    Memory capacity.

    OS

    Version of the operating system in use.

    Environment Variables

    Variable

    Names of environment variables.

    Value

    Values of environment variables.
