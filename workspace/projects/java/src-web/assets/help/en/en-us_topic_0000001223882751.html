Restarting a Guardian

Prerequisites

  • A Guardian is added.
  • The SSHD service has been enabled on the server where the Guardian is installed.

Procedure

  1. Log in to the Java Profiler WebUI, locate the Guardian to be modified, and click .
  2. In the dialog box that is displayed, enter the username and password for logging in to the remote server to redeploy a Guardian.

    If an error message indicating that the analysis assistant software cannot be installed is displayed when you deploy the Guardian, check whether the user name or password is correct and whether the current user is locked.
