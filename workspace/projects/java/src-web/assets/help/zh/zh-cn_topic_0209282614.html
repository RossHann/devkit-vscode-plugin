删除目标环境

前提条件

  • 已成功添加目标环境。
  • 安装分析辅助软件的服务器已开启sshd。

操作步骤

  1. 登录Java性能分析Web界面，在首页界面“目标环境列表”区域中选择指定目标环境，单击。
  2. 在弹出的删除操作确认对话框中单击“确认”。

    • 目标环境是离线状态时，删除目标环境需要输入部署分析辅助软件时的用户名和密码并单击“确认”。如果不输入密码直接单击“确认”，还需要手动删除远程服务器上分析辅助软件的执行文件。
    • 如果在删除目标环境过程中出现“无法安装分析辅助软件”的错误提示，请检查密码是否输入正确、当前用户是否被锁定。
