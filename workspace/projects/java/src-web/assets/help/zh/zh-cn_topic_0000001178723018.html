界面提示库文件安装失败

现象描述

安装库文件时，界面提示库文件安装失败。

可能原因

依赖库不匹配。

处理步骤

  • 如下类型库文件安装失败时，需要手动安装与操作系统版本匹配的库文件，然后再重新安装工具。
    • glibc
    • openssl
    • libffi-devel
  • 除以上3种类型库文件外，其他类型库文件安装失败时，需要手动安装与操作系统版本匹配的库文件，然后重新启动工具，重启工具的步骤请参见“常用操作 > 重启鲲鹏性能分析工具”。
