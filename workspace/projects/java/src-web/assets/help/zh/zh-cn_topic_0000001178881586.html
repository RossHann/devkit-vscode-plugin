查看GC分析结果

前提条件

已创建采样分析记录，并完成分析。

操作步骤

  1. 登录Java性能分析Web界面，单击首页界面“采样分析记录”列表中指定分析任务的名称。

    打开“概览”页签。

  2. 单击“GC”页签。

    查看GC分析结果，如图1、图2所示，参数说明如表1所示。

    点击按钮可查看当前任务的优化建议。
