操作日志管理

前提条件

已成功登录Java性能分析。

管理员用户（tunadmin）可以查看和下载所有用户的操作日志，普通用户只能查看和下载当前登录用户的操作日志。

操作步骤

  1. 在页面右上角菜单选项中单击“日志”。

    打开“操作日志”界面，如图1所示，界面参数描述如表1所示。
    • 以普通用户登录时，在页面右上角菜单选项中单击“操作日志”，可以打开“操作日志”界面。
    • 公共日志达到2000条会自动导出保存为“operation_logs.csv”文件，保存在“/opt/hyper_tuner/user_management/logs”目录下（“/opt/hyper_tuner”为工具安装目录，用户安装工具时可自定义），并清空“公共日志”页面的数据。当有新的“operation_logs.csv”文件生成时，会自动删除旧的“operation_logs.csv”文件。
    • Java性能分析操作日志默认保存30天，工具每天凌晨1点自动检查、删除超过30天的操作日志。
    图1 操作日志
    表1 参数说明

    参数

    说明

    操作用户

    显示进行操作的用户名称。

    操作名称

    显示操作事件的类型。

    操作结果

    显示操作的结果。

    操作主机IP

    显示进行操作的主机IP地址。

    操作时间

    显示操作发生的时间。

    操作详情

    显示操作的详细信息。

  2. 单击“下载日志”下载csv格式日志文件。
