查看快照信息

前提条件

已创建在线分析任务，且分析任务正在进行中。

操作步骤

  1. 登录Java性能分析Web界面，单击首页界面“在线分析记录”列表中指定分析任务的名称。

    打开“概览”页面。

  2. 在支持快照功能的页面，单击右侧的按钮保存当前页面的快照，快照的名称为保存快照的时间。

    如下页面支持保存快照，每个页面最多支持保存5份快照。
    • 内存转储
    • IO下的文件IO和Socket IO
    • 数据库下的JDBC、JDBC数据库连接池、MongoDB、Cassandra和HBase
    • HTTP请求

  3. 单击“快照”页签。

    查看保存的快照信息，如图1所示，参数描述请参见各页面对应的章节。

    使用IE浏览器时，快照对比信息表格不支持自定义列表项。

    图1 快照信息

    选中快照，单击图1左边区域的按钮可以删除快照。

    单击图1左边区域的按钮，选择快照类型和快照名称可以进行快照比对。
