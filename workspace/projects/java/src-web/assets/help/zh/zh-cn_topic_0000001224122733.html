查看概览信息

前提条件

已创建采样分析记录，并完成分析。

操作步骤

  1. 登录Java性能分析Web界面，单击首页界面“采样分析记录”列表中指定分析任务的名称。

    打开“概览”页签，如图1和图2所示，参数说明如表1所示。

    • “CPU利用率”区域显示了当前进程的CPU利用率折线图。
    • 点击按钮可查看当前任务的优化建议。
