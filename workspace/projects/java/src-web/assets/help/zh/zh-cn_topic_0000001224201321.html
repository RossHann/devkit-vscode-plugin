管理采样分析记录

前提条件

已成功创建采样分析记录并成功启动。

管理采样分析记录

登录Java性能分析Web界面，查看首页界面“采样分析记录”列表，如图1所示。

图1 采样分析记录

单击“采样分析记录”后面的，可以选择并导入采样分析记录文件。

单击分析任务后面的，可以导出在线分析记录文件，保存到本地。

单击分析任务后面的，可以删除采样分析记录文件。

删除分析任务会将该分析任务的历史数据都删除，请谨慎操作。

采样报告阈值配置

管理员可以修改采样报告阈值，普通用户只有查看权限。

  1. 在页面右上角的下拉菜单中，选择“报告阈值配置”。

    打开“报告阈值配置”界面，如图2所示。

    图2 报告阈值配置

  2. 单击“修改配置”。

    修改采样报告提示阈值、采样报告最大阈值。

    采样分析记录有数量限制，默认大于等于8份后会报警、默认大于等于15份后会提示客户删除旧记录。最大可配置为20份，达到采样报告最大阈值后，不删除旧记录，不能新建任务。

    历史报告达到提示阈值时如图3所示，达到最大阈值时如图4所示。

    图3 历史报告达到提示阈值
    图4 历史报告达到最大阈值
