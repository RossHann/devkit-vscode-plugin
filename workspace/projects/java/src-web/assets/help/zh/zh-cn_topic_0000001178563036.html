节点状态变为离线

现象描述

节点从在线状态变为离线状态。

可能原因

没有同步更新节点的server和agent证书。

处理步骤

  1. 重新生成证书。

    点击Agent服务证书界面里的生成证书按钮，重新生成证书。

  2. 确认节点。

    确认该离线节点是本机IP节点还是手动添加的节点。若为本机IP节点则需执行步骤3及步骤4，若为手动添加节点则只需要执行步骤4。

  3. 更换证书。

    更换离线节点对应的server证书，点击更换证书。

    更换离线节点对应的agent证书，点击更换证书。

  4. 确认节点状态。

    等待一段时间，节点状态会变为在线状态。
