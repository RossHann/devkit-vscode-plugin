配置操作系统安装依赖包

操作系统本地镜像源存在部分安装包缺失的情况，缺失的依赖包如表1所示，请自行在对应操作系统官方镜像源网站下载。将下载的安装包上传至服务器，执行如下命令安装依赖包。

  • 适用于CentOS操作系统：

    rpm -i 安装包名称

  • 适用于Debian/Ubuntu/UOS 20凝思操作系统：

    dpkg -i 安装包名称
