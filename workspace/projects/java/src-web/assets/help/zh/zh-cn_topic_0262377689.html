查看IO分析结果

前提条件

已创建采样分析记录，并完成分析。

操作步骤

  1. 登录Java性能分析Web界面，单击首页界面“采样分析记录”列表中指定分析任务的名称。
  2. 单击“IO”页签。

    打开“文件IO”页面，如图1所示，参数说明如表1所示。
