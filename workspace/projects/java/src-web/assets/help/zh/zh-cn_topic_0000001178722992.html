配置Docker容器环境

前提条件

  • 已在服务器上安装Docker容器。
  • 已获取对应操作系统版本的镜像文件。

操作步骤

以下步骤以openEuler为例，CentOS 7.6操作步骤相同。

  1. 使用SSH远程登录工具，将系统镜像文件上传至服务器任意目录下（例如“root”目录）。
  2. 使用SSH远程登录工具，以系统用户登录Linux操作系统命令行界面。

    该步骤必须在root用户下执行，若以普通系统用户登录，需要执行su切换到root用户下执行后续操作。

  3. 执行如下命令，将ISO镜像文件挂载到本地目录下（“root”为保存镜像文件的路径，请用实际路径代替）。

    mount /root/openEuler-20.03-LTS-everything-aarch64-dvd.iso /mnt

    重启后失效，需要重新挂载。

  4. 执行如下命令创建Docker容器：

    • host模式

      docker run -d --name containername -v /mnt/:/mnt/ --net=host imageid /usr/sbin/init

    • 端口映射模式

      docker run -d --name containername -v /mnt/:/mnt/ -p 8086:8086 --net=host imageid /usr/sbin/init

      • containername：自定义的Docker容器名称。
      • 第一个“/mnt/”：3中宿主机挂载对应宿主机操作系统镜像的目录，根据实际情况替换。
      • 第二个“/mnt/”：映射的容器的目录。
      • imageid ：系统镜像ID，可在宿主机执行docker images命令查询。
      • 端口映射模式中：8086为鲲鹏性能分析工具默认的HTTPS端口，需要工具安装过程中输入对应映射的容器端口号。

  5. 执行如下命令进入Docker容器：

    docker exec -it containerID /bin/bash

    containerID 为容器ID，可在宿主机执行docker ps命令查询。

  6. 配置Docker容器yum源。

    1. 执行如下命令备份原始yum源（mv命令中的“path”表示备份目录，请用实际情况代替）。

      cd /etc/yum.repos.d //进入yum源配置目录

      mv *.repo path //备份之前的yum源配置

    2. 配置yum源。
      1. 执行如下命令，创建“local.repo”文件。

        touch local.repo

      2. 执行如下命令修改“local.repo”文件中的内容。

        vi local.repo

      3. 按“i”进入编辑模式，在“local.repo”文件中写入以下内容。
        [local]
name=local repo
baseurl=file:///mnt
enabled=1
gpgcheck=0
      4. 按“ESC”，输入“:wq”，按“Enter”保存并退出。
    3. 执行如下命令，更新yum源。

      yum clean all

      yum makecache

  7. 执行如下命令安装依赖软件：

    • CentOS：

      yum install shadow passwd vim tar file

    • openEuler：

      yum install iproute shadow passwd vim tar

  8. 执行如下命令修改“/etc/hosts”文件中的内容。

    vi /etc/hosts

    在文件中添加如下内容（Docker IP请根据实际替换为容器的IP地址，可在Docker容器执行ip addr命令查询）：

    Docker IP  Malluma
