查看概览信息

前提条件

已创建在线分析任务，且分析任务正在进行中。

操作步骤

  1. 登录Java性能分析Web界面，单击首页界面“在线分析记录”列表中指定分析任务的名称。

    默认打开“概览”页面，如图1和图2所示，参数说明如表1所示。
    • “参数”区域显示该Java进程的系统参数，鼠标悬停于参数上方可看到详细信息。
    • 点击按钮可查看当前任务的优化建议。
    • “执行线程转储”按钮可转储当前线程状态，便于后续页签分析展示。
    • 点击按钮可导出当前全部页签的分析报告（内存、热点页签和GC日志、线程转储页签除外），如图3所示。
