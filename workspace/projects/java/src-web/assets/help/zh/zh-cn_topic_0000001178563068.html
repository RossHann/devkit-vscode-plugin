配置JDK

Java性能分析辅助软件采用非交互式Shell部署，依赖JDK 8及以上版本运行。需要在目标环境中安装支持版本的JDK，并配置 JAVA_HOME。

配置JAVA_HOME环境变量，可参考如下方法：

  • 对于Debian/Ubuntu/UOS 20/凝思操作系统，上传JDK文件至服务器中，在“ /etc/environment”文件中添加如下内容：

    export JAVA_HOME=/usr/jdk/jdk1.8.0

    其中 /usr/jdk/jdk1.8.0 是JDK文件的绝对路径，请根据实际场景填写。

    执行以下命令应用JAVA_HOME配置：

    source /etc/environment

  • 对于CentOS/EulerOS/openEuler/中标麒麟/麒麟V10/SUSE操作系统，上传JDK文件至服务器中，在“~/.bashrc”文件中配置JAVA_HOME，如果不存在“~/.bashrc”文件，可执行touch ~/.bashrc命令创建该文件，并在文件中添加如下内容：

    export JAVA_HOME=/usr/jdk/jdk1.8.0

    其中 /usr/jdk/jdk1.8.0 是JDK文件的绝对路径，请根据实际场景填写。

    执行以下命令应用JAVA_HOME配置：

    source ~/.bashrc

JAVA_HOME配置完成后，如果JDK版本是JDK 8，需要在CLASSPATH环境变量里配置正确的tools.jar路径。
