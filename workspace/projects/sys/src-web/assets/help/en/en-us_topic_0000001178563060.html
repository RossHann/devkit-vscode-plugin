Exporting a System Profiler Task

  1. Click the Export button.

    Figure 1 Exporting a task

  2. Select the project and task to be exported.

    Click OK to export the task. For details about the exported content, see Table 1.
    Figure 2 Export Task
    Table 1 Exported content

    Task Type

    Exported Content

    Overall analysis

    Log files, collected original data files, and database files

    Process/Thread performance analysis

    Log files, collected original data files, and database files

    Hotspot function analysis

    Log files, symbolic files, collected original data files, and database files

    Microarchitecture analysis

    Symbolic files, collected original data files, and database files

    Memory access analysis - Memory access statistics

    Log files, collected original data files, database files, and summary report files

    Memory access analysis - Miss events

    Log files, symbolic files, collected original data files, and database files

    Memory access analysis - False sharing

    Log files, symbolic files, collected original data files, and database files

    I/O analysis

    Log files, collected original data files, and database files

    Resource scheduling analysis

    Log files, collected original data files, and database files

    Locks and wait analysis

    Log files, symbolic files, collected original data files, and database files

    HPC analysis

    Log files, collected resolved intermediate files, and database files
