Project Management

Prerequisites

You have logged in to the System Profiler WebUI.

  • In general, big data, distributed storage, and database, a maximum of 10 nodes can be sampled and analyzed concurrently.
  • When you create an HPC task in an HPC project, all nodes (101 nodes at maximum) are selected by default. For other non-HPC tasks in an HPC project, 10 nodes are selected by default.

Procedure

  1. On the home page, click next to Project Management.

    The Create Project page is displayed.

  2. Set project parameters, as shown in Figure 1. For details about the parameters, see Table 1.

    Figure 1 Creating a project

    In the HPC scenario, sampling analysis can be performed on 100 nodes concurrently. You can click to search for node names and node IP addresses.

    Table 1 Parameter description

    Parameter

    Description

    Project Name

    Name of the project to be added. The name must meet the following requirements:

    • Contains only letters, digits, and special characters (including @#$ %^&*()[]<>._-!-+ and spaces).
    • Contain 1 to 32 characters.

    Scenario

    Select General Scenario, Big Data, Distributed Storage, HPC, or Database.

    Component
    • Spark2x: WordCount, TeraSort, and k-means
    • HBase: random read/write and BulkLoad
    • Hive: SQL1, SQL2, SQL3, SQL4, and SQL5
    • Elastic search: Elasticsearch cases
    • Flink: throughput
    • Kafka: producer throughput and consumer throughput
    • Redis: basic data type
    • Storm: storm case
      NOTE:

      This parameter is displayed when the Big Data scenario is selected.

    Storage Type
    • Ceph object storage: cold storage, balanced storage, and high-performance storage
    • Ceph block storage: balanced storage and high-performance storage
    • Ceph file storage: balanced storage
      NOTE:

      This parameter is displayed when the Distributed Storage scenario is selected.

    Database Type
    • MySQL 8
    • openGauss 2
      NOTE:

      This parameter is displayed when the Database scenario is selected.

    Select Nodes

    Node Name

    Name of a node.

    Node Status

    Status of the node.

    Node IP Address

    IP address of the server where the node is installed.

  3. Click OK.

    You can click the icons next to the project name to perform the following operations:

    • : creates an analysis task in the current project.
    • : modifies the project name and adds or deletes nodes for the project.
    • : deletes a project. After a project is deleted, all data related to the project will be deleted. Exercise caution when performing this operation.
    • The administrator (tunadmin) can view, modify, and delete projects created by all users.
    • Common users can view, modify, and delete their own projects and only view the projects created by other users.
    • After a project is created, you can click the project name to view details about the project.
