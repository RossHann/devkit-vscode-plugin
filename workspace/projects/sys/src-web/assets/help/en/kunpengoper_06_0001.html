Changing the IP Address and Port of the Kunpeng Hyper Tuner

In the following commands, /opt/hyper_tuner is the tool installation directory. Replace it with the actual directory.

  1. Use an SSH remote login tool to log in to the Linux CLI.
  2. Go to the tool installation path cd /opt/hyper_tuner and run the following command to change the IP address or service port.

    ./change_ip_port.sh {-ip/--ip=web server IP | -p/--port=web server port | -jip/--jip=java_profiler IP | -jp/--jport=java_profiler port}*

    Table 1 lists parameters in the command. Replace the IP address and port as required. If a port is occupied, change the port based on Changing an Internal Port.

    Example:

    ./change_ip_port.sh -ip=**.**.**.** -p=8087 -jip=**.**.**.** -jp=9091

    If the firewall has been automatically enabled during the tool installation, the port will be restored to the original status after the port is changed. The tool checks whether the modified port is enabled. If you type y on the interactive interface, the port is automatically enabled.

    Conventions for the command line formats:

    • Boldface: The keywords that must keep unchanged in a command line are in boldface.
    • { x | y | ... }*: Optional items are grouped in brackets and separated by vertical bars. A minimum of one item or a maximum of all items can be selected.
    Table 1 Parameter description

    Command

    Parameter

    Description

    -ip/--ip

    web server IP

    IP address for logging in to the Kunpeng Hyper Tuner WebUI.

    This parameter is optional.

    -p/--port

    web server port

    HTTPS port of the Kunpeng Hyper Tuner.

    This parameter is optional.

    Value range: 1024 to 65535

    Default value: 8086

    -jip/--jip

    java_profiler IP

    Internal IP address of the Java Profiler.

    This parameter is optional.

    -jp/--jport

    java_profiler port

    Internal communication port of the Java Profiler.

    This parameter is optional.

    Value range: 1024 to 65535

    Default value: 9090
    NOTE:
    • When configuring the IP address of the web server, enter the local IP address (Run ip addr on Linux to query the IP address).
    • If -ip/--ip and -jip/--jip are set to the same IP address, -p/--port and -jp/--jport must be different ports.

  3. Run the following command to start the Kunpeng Hyper Tuner:

    ./service.sh restart

    If the Kunpeng Hyper Tuner is installed on a Docker container, run the following command:

    ./service.sh restart -t docker
