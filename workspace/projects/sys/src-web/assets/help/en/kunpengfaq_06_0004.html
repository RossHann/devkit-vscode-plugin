Failed to Perform a System Resource Scheduling Analysis Task (Attach to Process Type)

Symptom

During the creation of a system resource scheduling analysis task, the task type is set to Attach to Process and the PID is set to the OS process ID. As a result, the task fails. The following error information is displayed in data analysis logs.

Possible Causes

If the Linux kernel version is earlier than V4.15-rc1, data collection may fail during system resource scheduling analysis. As a result, the task fails.

Procedure

Upgrade the kernel version to V4.15-rc1.
