Using the PGO Optimizer for Compilation

Symptom

After a microarchitecture task is created, a message "Use PGO for compilation." is displayed when you view the Summary tab page of the analysis result.

Possible Causes

The PGO optimizer is needed for compilation.

Profile-Guided Optimization (PGO) improves application performance by reducing cache-thrashing, reorganizing code layout, reducing code length, and reducing branch mispredictions.

Procedure

  1. Add the -fprofile-generate=<profile_dir> option during compilation.
  2. Compile and run the software, and collect configuration files.
  3. Recompile the configuration files collected in 2 with the option -fprofile-use=<profile_dir> to complete the compilation with the PGO.
