Importing the Root Certificate for the Kunpeng Hyper Tuner

If a security alert is displayed when you use a browser to log in to the Kunpeng Hyper Tuner, you can import the root certificate for the tool to shield the security alert.

  1. Log in to Huawei PKI Download Management and choose Level-2 CA > IT, as shown in Figure 1.

    Figure 1 Choosing IT

  2. Download the certificate whose code is P7B and save it to a local folder, as shown in Figure 2.

    Figure 2 Downloading the root certificate

  3. Open the Chrome browser, choose Settings > Advanced > Privacy and security > Manage certificates, as shown in Figure 3.

    Figure 3 Privacy and security

  4. In the displayed Certificates dialog box, select the Trusted Root Certification Authorities tab and click Import, as shown in Figure 4.

    Figure 4 Certificates

  5. In the displayed Certificate Import Wizard dialog box, click Next, as shown in Figure 5.

    Figure 5 Welcome to the Certificate Import Wizard

  6. On the page for selecting the certificate, click Browse, as shown in Figure 6.

    Figure 6 Importing the certificate

  7. Select the certificate to be imported.
  8. Click Next.

    On the page for selecting the position where the certificate will be saved, select the position shown in Figure 7.

    Figure 7 Selecting position for saving the certificate

  9. Click Next and then Finish.

    The Security Warning dialog box is displayed.

  10. Click Yes.

    The message "The import was successful" is displayed.

  11. Click OK to complete the import.
  12. Modify the mapping between the server IP address and the tool name in the local hosts file.

    1. Open the hosts file in the C:\Windows\System32\drivers\etc path.
    2. Add the following content to the hosts file, save the changes, and exit.

      Server IP address Hyper-tuner

      Example: 10.175.119.171 Hyper-tuner

  13. Open the browser, enter https://hyper-tuner:Port number (for example: https://hyper-tuner:8086) in the address bar, and click Enter.
