Changing an Internal Port

Changing Port 8080

  1. Modify the THOR_WEB_SERVER_PORT parameter in the thor.properties file.

    cd /opt/hyper_tuner/thor-server-*/config

    Modify THOR_WEB_SERVER_PORT=8080 in the thor.properties file. Change the port from 8080 to another port that is not listened on.

  2. Modify the proxy_pass parameter in the nginx.conf file.

    cd /opt/hyper_tuner/tool/nginx/conf

    Modify proxy_pass http://127.0.0.1:8080 in the nginx.conf file. Change the port 8080 to the port you specify in 1.

  3. Modify the THOR_WEB_SERVER_PORT parameter in the install.ini file.

    cd /opt/hyper_tuner/

    Modify THOR_WEB_SERVER_PORT=8080 in the install.ini file. Change the port 8080 to the port you specify in 1.

  4. Restart the service.

    systemctl restart hyper_tuner_nginx

    systemctl restart gunicorn_user

    Modify proxy_pass http://127.0.0.1:8000 in the nginx.conf file. Change the port from 8000 to another port that is not listened on.

    Modify GUNICORN_PORT=8000 in the install.ini file. Change the port 8000 to the port you specify in 1.

    systemctl restart hyper_tuner_nginx

    systemctl restart gunicorn_user
