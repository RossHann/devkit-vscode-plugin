Welcome to the System Profiler Help

The System Profiler is a tool of the Kunpeng Hyper Tuner.

The System Profiler is a performance analysis and tuning tool for servers powered by Kunpeng processors. It collects performance data of processor hardware, operating system, processes/threads, and functions, analyzes system performance indicators, locates system bottlenecks and hotspot functions, and provides tuning suggestions. This tool helps you quickly locate and handle software performance problems.

This Help provides the operation guide and precautions for using the System Profiler.
