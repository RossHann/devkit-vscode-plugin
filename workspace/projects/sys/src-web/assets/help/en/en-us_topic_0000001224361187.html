Weak Password Dictionary Management

A weak password can be easily guessed or cracked. When a password is set or change, if the new password has a match in the weak password dictionary, the user will be forced to change the password for security purposes.

Prerequisites

You have logged in to the Kunpeng Hyper Tuner or System Profiler WebUI.

  • The administrator (tunadmin) can add and delete weak passwords. Common users can only view weak passwords.
  • A maximum of 1000 weak passwords can be added.

Procedure

  1. Click in the upper right corner and choose Weak Password Dictionary.

    The Weak Password Dictionary page is displayed.

  2. Click Add.

    The Add Weak Password dialog box is displayed.

  3. Enter the weak password to be added in the text box. The weak password must meet the following requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of the following characters:
      • Uppercase letters A to Z
      • Lowercase letters a to z
      • Digits 0 to 9
      • Special characters (`~!@#$%^&*()-_=+\|[{}];:'",<.>/?)

  4. Click OK.

    To delete a weak password, locate the weak password and click Delete in the Operation column. In the Delete Weak Password dialog box displayed, click OK.
