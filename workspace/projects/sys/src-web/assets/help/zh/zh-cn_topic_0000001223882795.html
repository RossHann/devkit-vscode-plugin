热点函数性能分析/资源调度分析结果中的代码流区域显示不支持该功能

现象描述

热点函数性能分析/资源调度分析结果中的代码流区域提示“本系统无graphviz组件，不支持该功能”，如图1所示。

图1 代码流区域

可能原因

操作系统中未安装graphviz组件。

处理步骤

配置操作系统本地镜像源或网络镜像源安装graphviz组件。

安装高版本的graphviz可能会导致部分系统库组件（如libc6和libdb）升级到高版本，高版本的系统库组件可能影响系统的其他功能和软件运行，如导致systemd运行异常，请根据自己的运行环境来确定是否需要安装升级graphviz组件。

安装步骤如下（以CentOS 7.6安装graphviz组件为例）：

  1. 执行如下命令安装graphviz组件。

    yum -y install graphviz

  2. 执行如下命令确认安装成功。

    rpm -qa | grep graphviz

    显示如下内容说明安装成功。

    graphviz-2.30.1-21.el7.aarch64
