EulerOS系统上动态库热点函数关联汇编指令失败

现象描述

资源调度分析和热点函数性能分析任务执行成功后，查看函数对应源码和汇编指令，页面显示无数据。

可能原因

在EulerOS系统上，鲲鹏性能分析工具程序的操作系统运行用户（malluma）对objdump和readelf命令无可执行权限，导致关联函数的起始地址以及终止地址失败。

处理步骤

以root用户登录服务器增加malluma用户对objdump和readelf命令的可执行权限。

  1. 执行如下命令查看objdump和readelf可执行文件所在的位置。

    which objdump

    which readelf

  2. 执行如下命令增加malluma用户的可执行权限。

    chmod o+x /usr/bin/objdump

    chmod o+x /usr/bin/readelf
