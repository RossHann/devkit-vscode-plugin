以命令行方式执行安装或卸载脚本

安装

鲲鹏性能分析工具安装脚本的命令格式如下，参数说明如表1所示。

./install.sh [-i | -r | -u] [-a | -j | -s | -c | -l] [-d=install path] -ip=web server IP [-p=web server port] [-mip=mallumad IP] [-mp=mallumad port] [-jip=java-perf IP] [-jp=java-perf port] -jh=java home

  • 如果只安装系统性能分析，安装命令不涉及-jip、和-jp和-jh。
  • 如果只安装Java性能分析，安装命令不涉及-mip和-mp。
  • 命令行格式的约定：

    粗体：命令行关键字（命令中保持不变、必须照输的部分）采用加粗字体表示。

    斜体：命令行参数（命令中必须由实际值进行替代的部分）采用斜体表示。

    [ ]：表示用“[ ]”括起来的部分在命令配置时是可选的。

    [ x | y | ... ]：表示从两个或多个选项中选取一个或者不选。

表1 安装命令参数说明

命令

参数选项

说明

-i/--install

-r/--rollback

-u/--upgrade

-l

-c

-

可选参数，选择操作类型。默认为“-i/--install”。可选择：

  • -i/--install：表示选择安装工具。
  • -r/--rollback：表示选择回退工具。
  • -u/--upgrade：表示选择升级工具。
  • -l：表示安装前进行依赖项检查。
  • -c：表示清除之前安装的工具。

-a/--all

-j/--java

-s/--system

-

可选参数，选择安装的工具类型。默认为“-a/--all”。可选择：

  • -a/--all：安装系统性能分析和Java性能分析。
  • -j/--java：只安装Java性能分析。
  • -s/--system：只安装系统性能分析。

-d/--directory

install path

可选参数，设置安装路径。默认为安装在“/opt”目录下。

-ip/--ip

web server IP

必配参数，表示安装工具的服务器IP地址。

-p/--port

web server port

可选参数，表示鲲鹏性能分析工具的HTTPS端口。默认为8086，设置范围为1024~65535。

-mip/--mip

mallumad IP

可选参数，表示系统性能分析内部使用的IP。默认为使用“-ip/--ip”参数设置的“web server IP”。

-mp/--mport

mallumad port

可选参数，表示Web服务器（即安装系统性能分析的服务器）和Agent节点服务器之间的通信端口。默认为50051，设置范围为1024~65535。

-jip/--jip

java-perf IP

安装Java性能分析时可选参数，表示Java性能分析内部使用的IP。默认为使用“-ip/--ip”参数设置的“web server IP”。

-jp/--jport

java-perf port

安装Java性能分析时可选参数，表示Java性能分析的内部模块通信端口。默认为9090，设置范围为1024~65535。

-jh/--java_home

java home

安装Java性能分析时可选参数，表示服务器中安装JDK的路径（JDK工具版本要求为Open JDK 11），可以通过echo $JAVA_HOME命令获取。
说明：
  • 配置IP地址时，必须输入本机IP地址（Linux操作系统下，可以通过ip addr命令获取）。
  • 如果“-ip/--ip”、“-mip/--mip”或“-jip/--jip”设置的IP地址相同，“-p/--port”、“-mp/--mport”或“-jp/--jport”不能设置为同一个端口。
  • 如果与系统已占用的端口冲突，会自动+1。循环100次未找到可用端口将终止安装。

例如：执行如下命令安装系统性能分析和Java性能分析：

./install.sh -i -a -d=/opt -ip=10.247.81.189 -p=8086 -mip=10.247.81.189 -mp=50051 -jip=10.247.81.189 -jp=9090 -jh=/opt/jdk11

或者省略为：

./install.sh -ip=10.247.81.189 -jh=/opt/jdk11

卸载

鲲鹏性能分析工具卸载脚本的命令格式如下，参数说明如表2所示。

./hyper_tuner_uninstall.sh [-a | -j | -s]

命令行格式的约定：

  • 粗体：命令行关键字（命令中保持不变、必须照输的部分）采用加粗字体表示。
  • [ x | y | ... ]：表示从两个或多个选项中选取一个或者不选。
表2 卸载命令参数说明

命令

参数选项

说明

-a/--all

-j/--java

-s/--system

-

可选参数，选择卸载的工具类型。默认为“-a/--all”。可设置：

  • -a/--all：卸载系统性能分析和Java性能分析。
  • -j/--java：卸载Java性能分析。
  • -s/--system：卸载系统性能分析。

例如：执行如下命令卸载系统性能分析和Java性能分析：

./hyper_tuner_uninstall.sh -a

或者省略为：

./hyper_tuner_uninstall.sh
