修改内部端口号

8080端口的修改方法

  1. 修改thor.properties文件中THOR_WEB_SERVER_PORT参数。

    cd /opt/hyper_tuner/thor-server-*/config

    修改thor.properties中THOR_WEB_SERVER_PORT=8080 。修改8080端口号为其它未被监听的端口号。

  2. 修改nginx.conf文件proxy_pass参数。

    cd /opt/hyper_tuner/tool/nginx/conf

    修改nginx.conf 中 proxy_pass http://127.0.0.1:8080 。修改8080端口号，与1修改的端口号一致。

  3. 修改install.ini文件的THOR_WEB_SERVER_PORT参数。

    cd /opt/hyper_tuner/

    修改install.ini中THOR_WEB_SERVER_PORT=8080。修改8080端口号，与1修改端口号一致。

  4. 重启服务。

    systemctl restart hyper_tuner_nginx

    systemctl restart gunicorn_user

8000端口的修改方法

    cd /opt/hyper_tuner/tool/nginx/conf

    修改nginx.conf 中proxy_pass http://127.0.0.1:8000 。修改8000端口号为其它未被监听的端口号。

  2. 修改gunicorn.py.ini文件的bind参数。

    cd /opt/hyper_tuner/user_management/visualization/Django/user_management/user_management

    修改gunicorn.py.ini中bind = '127.0.0.1:8000'。修改8000端口，与1修改端口号一致。

  3. 修改install.ini文件中GUNICORN_PORT参数。

    cd /opt/hyper_tuner/

    修改install.ini中GUNICORN_PORT=8000。修改8000端口号，与1修改端口号一致。

  4. 重启服务。

    systemctl restart hyper_tuner_nginx

    systemctl restart gunicorn_user
