Editing a Password

Prerequisites

You have logged in to Kunpeng Hyper Tuner or System Diagnosis.

Procedure

  1. Open the login user drop-down list box in the upper right corner and choose Edit Password.

    The Edit Password dialog box is displayed, as shown in Figure 1. For details about the parameters, see Table 1.
    Figure 1 Editing a password
    Table 1 Parameters for editing a password

    Parameter

    Description

    Old Password

    Old password of the user.

    New Password

    New password to be used. The password must meet the following complexity requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of uppercase letters, lowercase letters, digits, and special characters (`~!@#$%^&*()-_=+\|[{}];:'",<.>/?).
    • Cannot be the same as the user name or the user name in reverse order.
    • Cannot be the old password in reverse order.
    • Cannot be a password in the weak password dictionary.

    Confirm Password

    Password re-entered for confirmation.

  2. Edit the password by referring to Table 1 and click OK.
