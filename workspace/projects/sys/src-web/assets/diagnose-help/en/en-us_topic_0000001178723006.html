Login

This section describes how to log in to the System Diagnosis WebUI, using Internet Explorer 11.0 on Windows 10 as an example.

Browser Requirements

Table 1 lists the browser versions supported by the System Profiler.

Table 1 Browser requirements

Browser

Version

Chrome

Google Chrome 72.0 or later

Internet Explorer 11.0 or later

Edge

Microsoft Edge 79.0 or later

Procedure

  • By default, the system locks a user account upon five consecutive login failures. The user account will be automatically unlocked three minutes later.
  • If an IP address fails to log in to the system for 10 consecutive times within 30 seconds, the IP address will be locked for 30 minutes.
  • The default system timeout period is 30 minutes. If no operation is performed on the WebUI within 30 minutes, the user will be automatically logged out of the WebUI. The administrator can click and choose System Settings on the Kunpeng Hyper Tuner or System Diagnosis WebUI to manually set Session Timeout Period (min). The value ranges from 10 to 240 minutes.
  • By default, a maximum of 10 common users are allowed to log in to the system at the same time. The number of administrators who can concurrently use this tool is not restricted. The administrator can click and choose System Settings on the Kunpeng Hyper Tuner or System Diagnosis WebUI to set Maximum Number of Online Common Users. The value range is 1 to 20.
  • Only one active session is allowed for a user. If a user has logged in to the tool, re-login will terminate the previous session established.
  1. Open a browser on the local PC, enter https://Server IP address:Port number in the address box, and press Enter.

    The security alert window is displayed, as shown in Figure 1.
    Figure 1 Security alert
    • Port number is the HTTPS port number (8086 by default) set during the tool installation. Change the port number based on the site requirements. Before using this tool, ensure that HTTPS port is enabled on the operating system firewall.
    • You can also enter https://Server IP address:Port number/user-management/#/login and press Enter to open login page.
    • If a website security alert is displayed, you can ignore this message or download the Huawei root certificate to shield this alert. For details, see Importing the Root Certificate for the Kunpeng Hyper Tuner.

  2. Click Continue to this website (not recommended).

    The Kunpeng Hyper Tuner login page is displayed, as shown in Figure 2. Table 2 describes the parameters.

    You can view the online help from the drop-down list of in the upper right corner.

    Figure 2 Kunpeng Hyper Tuner login page (first login)
    Table 2 Parameters on the first user login page

    Parameter

    Description

    User Name

    User name for logging in to the Kunpeng Hyper Tuner WebUI.

    NOTE:

    The default system administrator is tunadmin.

    Password

    Password of a user for logging in to the tool. The password must meet the following complexity requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of uppercase letters, lowercase letters, digits, and special characters (`~!@#$%^&*()-_=+\|[{}];:'",<.>/?).
    • Cannot be the same as the user name or the user name in reverse order.
    • Cannot be a password in the weak password dictionary.
      NOTE:

      After the tool is installed, set the administrator password upon the first login.

    Confirm Password

    Password re-entered for confirmation.
    • The password of the Kunpeng Hyper Tuner user is valid for 90 days by default. You are advised to change the password before the password expires. If the password has expired, change the password after logging in to the system using the old one. Otherwise, you cannot perform the next step. Note that the new password cannot be the old password in reverse order.
    • The administrator can choose System Settings from the menu options and manually set password validity period. A password is valid for 7 to 90 days.

    Set the administrator password and click OK.

    The login page is displayed, as shown in Figure 3.

    Figure 3 Kunpeng Hyper Tuner login page

  3. Enter the user name and password, and click Log In.

    If the login is successful, the Kunpeng Hyper Tuner home page is displayed, as shown in Figure 4, and the user name is displayed in the upper right corner of the page.

    Figure 4 Kunpeng Hyper Tuner home page

  4. Click System Diagnosis.

    The System Diagnosis page is displayed, as shown in Figure 5.

    Upon your first login, the Disclaimer page will be displayed. Read it carefully, select "I have read the disclaimer", and click Agree. You can click in the upper right corner and choose Disclaimer to view the disclaimer again.

    Figure 5 Home page of System Diagnosis
