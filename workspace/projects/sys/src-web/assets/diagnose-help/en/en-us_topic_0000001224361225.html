Configuring the OS Installation Dependencies

Some installation packages are missing from the local image source of the OS. For details about the missing dependency packages, see Table 1. Download the image from the official website of the corresponding OS. Upload the downloaded dependency package to the server and run the following command to install the dependency package.

  • CentOS

    rpm -i Installation package name

  • Debian/Ubuntu/UOS 20/Linx

    dpkg -i Installation package name
