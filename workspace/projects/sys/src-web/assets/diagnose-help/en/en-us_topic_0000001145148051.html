Welcome to the System Diagnosis Help

System Diagnosis is a tool of Kunpeng Hyper Tuner.

System Diagnosis provides memory leak diagnosis (including memory release failure and abnormal memory release), abnormal memory access diagnosis, memory consumption information analysis and display, and out of memory (OOM) diagnosis capabilities. In this way, it helps you identify memory usage issues in source code and improve program reliability.

This Help provides the operation guide and precautions for using the System Diagnosis.
