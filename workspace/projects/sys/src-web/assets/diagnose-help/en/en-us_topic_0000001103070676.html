OOM

Prerequisites

The OOM has been selected for the diagnosis content, a memory leak diagnosis task has been created, and the analysis has been completed.

Procedure

  1. In the Project Management area, click before the target project and analysis task.

    The node list is displayed.

  2. Click the name of the target node to view the analysis result.

    Click the node name and click the OOM tab. See Figure 1.
    Figure 1 OOM
    Table 1 OOM parameters

    Parameter

    Description

    Basic Information

    Time points, triggers, and murdered processes.

    Call Stack

    Call stack information.

    System Memory

    System memory information.

    Process Memory

    Process memory information.

  3. Click View Details to view OOM details. See Figure 2.

    Figure 2 OOM details

  4. Click the Task Information tab to view the detailed configuration and sampling information about the task on the current node.

    If the task fails to be executed, the failure cause is displayed on the Task Information tab page.

  5. Click the Task Logs tab page to view the detailed log information.
