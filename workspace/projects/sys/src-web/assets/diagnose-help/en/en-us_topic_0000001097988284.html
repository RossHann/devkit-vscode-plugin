WebUI Description

Kunpeng Hyper Tuner Home Page

Figure 1 shows the home page of the Kunpeng Hyper Tuner. Table 1 describes the areas on the home page.

Figure 1 Kunpeng Hyper Tuner home page
Table 1 Areas on the Kunpeng Hyper Tuner home page

Area

Element

Description

1

System Profiler

Provides an entry to the System Profiler WebUI.

2

Java Profiler

Provides an entry to the Java Profiler WebUI.

3

System Diagnosis

Navigation entry to System Diagnosis.

4

Tuning Assistant

Provides an entry to access the Tuning Assistant.

5

Current user

Displays the current login user, and provides an entry for changing the user password and logging out.

6

Settings

Provides navigation paths to user management, weak password dictionary management, system settings, public log management, and web server certificate management.

7

More

Provides an entry for changing the language between simplified Chinese and English, providing feedback, and viewing Kunpeng Hyper Tuner release information.

8

Feedback

Navigation entry for feedback.

The System Profiler, Java Profiler and System Diagnosis WebUIs are subject to change.

Areas on the System Diagnosis page

Figure 2 shows the home page of the System Diagnosis. Table 2 describes the areas on the home page.

Figure 2 Home page of System Diagnosis
Table 2 Home page of System Diagnosis

Area

Element

Description

1

Project information area

Displays all the projects and tasks created in the System Diagnosis and provides a button for creating a project.

2

New task area

Displays the parameters for creating a task and analysis results and provides a button for creating a task.

3

Current user

Displays the current login user, and provides an entry for changing the user password and logging out.

4

Back to Home

Returns to the home page of the Kunpeng Hyper Tuner.

NOTE:

You can also click the tool icon in the upper left corner of the page to return to the home page.

5

Server management

Provides an entry for node management and agent service certificate management.

6

Settings

Provides navigation paths to user management, weak password dictionary management, system settings, scheduled task management, task import/export management, task template management, and log management.

7

More

Provides an entry for changing the language between simplified Chinese and English, providing feedback, viewing the wizard, online help, disclaimer, and viewing release information.

8

Kunpeng Community

Provides the latest information about Kunpeng.

9

Feedback

Navigation entry for feedback.
