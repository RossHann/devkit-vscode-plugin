Project Management

Prerequisites

You have logged in to the Tuning Assistant.

Procedure

  1. On the home page, click next to Project Management.

    The Create Project page is displayed.

  2. Set project parameters, as shown in Figure 1. For details about the parameters, see Table 1.

    Figure 1 Creating a project
    Table 1 Parameter description

    Parameter

    Description

    Project Name

    Name of the project to be added. The name must meet the following requirements:

    • Contain only letters, digits, and special characters (including @#$%^&*()[]<>._-!~+ and spaces).
    • Contain 1 to 32 characters.

    Select Nodes

    Node Name

    Name of a node.

    Node Status

    Status of the node.

    Node IP Address

    IP address of the server where the node is installed.

  3. Click OK.

    You can click the icons next to the project name to perform the following operations:

    • : creates an analysis task in the current project.
    • : deletes a project. After a project is deleted, all data related to the project will be deleted. Exercise caution when performing this operation.
    • The administrator (tunadmin) can view and delete projects created by all users.
    • Common users can view and delete their own projects and only view the projects created by other users.
    • After a project is created, you can click the project name to view details about the project.
