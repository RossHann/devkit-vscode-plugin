Area

Description

Project information area

Displays all the projects and analysis tasks created in the Tuning Assistant and provides a button for creating a project.

Provides an entry for switching the language between simplified Chinese and English, providing feedback, and viewing the wizard, online help, disclaimer, and the Tuning Assistant release information.

Provides navigation paths to user management, weak password dictionary management, system settings, and log management.

Provides an entry for managing nodes.

Back to home

Returns to the home page of the Kunpeng Hyper Tuner.

NOTE:

You can also click the tool icon in the upper left corner of the page to return to the Kunpeng Hyper Tuner home page.

7

Current user

Displays the current login user, and provides an entry for changing the user password and logging out.

8

Case links

Provides links to typical cases and sample code for each function.

The drive capacity monitoring icon is displayed in the lower right corner of the page for prompting alarms and clearance information when one the following conditions occurs:

  • If the remaining drive capacity is less than 10% of the maximum capacity or the drive capacity used by the tool reaches 80% of the maximum capacity, an alarm highlighted in red is generated and a monitoring icon is displayed on the page.
  • If the drive capacity used by the tool reaches 90%, the remaining drive capacity is less than 5% of the maximum capacity, or the remaining drive capacity of the server is less than 5 GB, the system can only perform query and deletion operations. The running tasks will be stopped, and new tasks will fail to be created.
  • If the remaining drive capacity is greater than 25% of the maximum capacity and the capacity used by the tool is less than 75% of the maximum capacity, the alarm icon is displayed in yellow.
