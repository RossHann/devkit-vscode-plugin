Viewing Analysis Results

Prerequisites

A comparison analysis task has been created.

Procedure

  1. In the Analysis Task Management area on the left, click the target analysis report. The analysis report page is displayed.

    Figure 1 Analysis report page

  2. You can click options in the top area on the analysis report to view the analysis results of system configurations, hotspot functions, system performance, process/thread performance, and object information.
