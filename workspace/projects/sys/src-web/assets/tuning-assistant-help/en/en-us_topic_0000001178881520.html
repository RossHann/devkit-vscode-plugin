Creating a Comparison Analysis Task

Prerequisites

  • A common analysis project has been created, and the project does not contain a node in the Offline state.
  • Users can create analysis tasks only under their own projects.

Procedure

  1. Click next to Analysis Report Management.

    Create a comparison analysis task. The system provides two analysis modes, as shown in Table 1.

    Table 1 Comparison analysis task mode

    Analysis Mode

    Description

    Horizontal Analysis

    Compares the analysis results of multiple nodes under the same task of the same project.

    Vertical Analysis

    Compares the analysis results of two nodes of different tasks.

  2. (Optional) Create a comparison analysis task and select the Horizontal Analysis mode.

    1. Select different nodes under the same task of the same project. Click to change the positions of object 1 and object 2.
      Figure 1 Creating a horizontal comparison analysis task
    2. Click OK. The horizontal comparison analysis task is created.

  3. (Optional) Create a comparison analysis task and select the Vertical Analysis mode.

    1. Select two nodes under different tasks. Click to change the positions of object 1 and object 2.
      Figure 2 Creating a vertical comparison analysis task
    2. Click OK. The vertical comparison analysis task is created.

  4. (Optional) Delete historical reports.

    After a historical report is deleted, the analysis data of the historical report cannot be restored. Exercise caution when performing this operation.

    1. Select a history report under Analysis Report Management in the left pane. Click on the right of the selected history report.
      Figure 3 Deleting a report
    2. Click OK to delete the history report.
