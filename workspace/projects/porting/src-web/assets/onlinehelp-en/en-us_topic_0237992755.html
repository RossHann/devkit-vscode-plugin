Operation Log Management

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

  • Common users can only view and download their own operation logs. The administrator can view operation logs of all users.
  • When the number of operation log records reaches 2000, the tool generates a his_operation_log.csv file in the /opt/portadv/logs directory (/opt/portadv is the tool installation directory, which can be customized during the tool installation), and clears data on the Operation Log page. When a new his_operation_log.csv file is generated, the old file will be automatically overwritten.
  • The operation logs will be retained for 30 days by default. The tool automatically checks operation logs at 1:00 every day and deletes expired log records.

Procedure

  1. On the WebUI, click in the upper right corner and choose Log.

    On the Log page displayed, choose Operation Log, as shown in Figure 1.
    Figure 1 Operation log

  2. (Optional) Download the operation logs.

    Click Download to download the log file in CSV format.
