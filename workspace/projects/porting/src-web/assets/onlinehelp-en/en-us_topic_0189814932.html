Deleting a User

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

  • Only the administrator portadmin can delete users.
  • The default administrator cannot be deleted.

Procedure

  1. On the WebUI, click in the upper right corner and choose Manage Users.

    The user management page is displayed.

  2. Locate the user to be deleted, and click Delete in the Operation column.

    After a user is deleted, all historical data of the user will be deleted. Exercise caution when performing this operation.

  3. Enter the password of the administrator and click OK.
