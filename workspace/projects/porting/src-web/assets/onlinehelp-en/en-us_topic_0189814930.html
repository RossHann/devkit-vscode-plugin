Creating a User

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

Only the administrator portadmin can create users.

Procedure

  1. Click in the upper right corner and choose Manage Users.

    The user management page is displayed.

  2. Click Add.

    The Add User page is displayed, as shown in Figure 1. Table 1 describes the parameters.

    Figure 1 Add User page
    Table 1 Parameter description

    Parameter

    Description

    User Name

    Name of the user to be added. The user name must meet the following requirements:

    • Start with an English letter.
    • Contain 6 to 32 characters.
    • Allow letters, digits, hyphens (-), and underscores (_).

    Workspace

    Root directory to which the code is uploaded.

    This parameter is dimmed and cannot be modified.

    Administrator Password

    Password of the administrator.

    Password

    Password of the user to be added. The password must meet the following complexity requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of uppercase letters, lowercase letters, digits, and special characters (`~!@#$%^&*()-_=+\|[{}];:'",<.>/?).
    • Cannot contain any space.
    • Cannot be the same as the user name.
    • Cannot be a password in the weak password dictionary.

    Confirm Password

    Password re-entered for confirmation.

  3. Set parameters (see Table 1) and click OK.

    • There is no limit on the number of common users.
    • Common users can perform the following operations:
      • Perform software porting assessment, source code porting, software package rebuilding, and dedicated software porting, and use enhanced functions provided by the tool.
      • View and download operation logs.
      • Configure parameters for scanning.
      • View web server certificate information.
      • View and search for weak passwords.
      • View, download, and delete their own reports.
