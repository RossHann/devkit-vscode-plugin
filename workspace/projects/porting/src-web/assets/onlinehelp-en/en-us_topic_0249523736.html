Enhanced Functions

The tool provides the following enhanced functions:

  • 64-bit running mode check (for x86 and Kunpeng platforms): Pre-checks the 32-bit applications to be ported to the Kunpeng platform. GCC 4.8.5 to GCC 9.3.0 are supported.
  • Structure byte alignment check (for x86 and Kunpeng platforms): Provides information about byte alignment of structure variables.
  • Cache line alignment check (for x86 and Kunpeng platforms): Provides information about cache line alignment.
  • Memory consistency check (for the Kunpeng platform only): Checks for memory consistency issues in source code.
  • 64-bit Mode Check
