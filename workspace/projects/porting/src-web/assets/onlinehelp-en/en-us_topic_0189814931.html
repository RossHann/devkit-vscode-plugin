Resetting a User Password

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

Only the administrator portadmin can reset passwords.

Procedure

  1. On the WebUI, click in the upper right corner and choose Manage Users.

    The user management page is displayed.

  2. Locate the user whose password needs to be reset, and click Reset Password.

    The Reset Password page is displayed, as shown in Figure 1. Table 1 describes the parameters.

    Figure 1 Reset Password
    Table 1 Parameter description

    Parameter

    Description

    User Name

    Name of the user whose password needs to be reset. The user name cannot be changed when the password is reset.

    Workspace

    Root directory of the code uploaded by the user. The workspace cannot be changed when the password is reset.

    Administrator Password

    Password of the administrator.

    Password

    New password to be used. The password must meet the following complexity requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of uppercase letters, lowercase letters, digits, and special characters (`~!@#$%^&*()-_=+\|[{}];:'",<.>/?).
    • Cannot contain any space.
    • Cannot be the same as the user name.
    • Cannot be a password in the weak password dictionary.

    Confirm Password

    Password re-entered for confirmation.

  3. Reset the password (see Table 1) and click OK.

    If the password of a logged-in user is reset, the logged-in user will automatically exit to the login page. The user needs to use the new password to log in to the system again.
