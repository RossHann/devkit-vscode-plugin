Downloading an Analysis Report

Prerequisites

The analysis task is complete.

Downloading an Analysis Report in the Historical Report Area

  1. On the WebUI, choose Source Code Porting from the navigation pane on the left.
  2. In the Historical Reports area, locate the report to be downloaded and click , as shown in Figure 1.

    Figure 1 Historical Reports

  3. Click Download Report (.csv) or Download Report (.html).

Downloading an Analysis Report on the Porting Report Page

  1. On the WebUI, choose Source Code Porting from the navigation pane on the left.
  2. In the list of historical reports, click the report name of the analysis task.

    The report page is displayed.

    Click View Report. The Porting Report page is displayed.

  3. Click Download Report (.csv) or Download Report (.html) to download the report.
