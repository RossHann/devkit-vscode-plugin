Welcome to Kunpeng Porting Advisor Online Help

Kunpeng Porting Advisor is an application porting tool developed for developers, users, and independent software vendors (ISVs) of the Kunpeng platform. This tool can be installed on x86 servers or servers powered by Kunpeng 916 or 920 processors. It provides the following functions:

  • Software porting assessment: The tool scans and analyzes software packages (not source code packages) and installed software, and provides porting assessment reports.
  • Source code porting: The tool checks and analyzes source code files in C/C++, ASM, Fortran, interpreted language, and assembly files. It also identifies the code to be ported, and provides porting suggestions. Editing and one-click code replacement are supported.
  • Software package rebuilding: The tool analyzes the composition of the software package to be ported on the Kunpeng platform and rebuilds a software package compatible with the Kunpeng platform or provides a software package that can be directly used.
  • Dedicated software porting: The tool modifies the source code of common solutions on the Kunpeng platform, compiles the code, and builds the software packages compatible with the Kunpeng platform.
  • Enhanced functions: The tool can check 64-bit running mode, structure byte alignment, and cache line alignment on x86 and Kunpeng platforms. It can also check memory consistency on the Kunpeng platform.
    • 64-bit running mode check: The tool identifies the applications to be ported from the 32-bit platform to the 64-bit platform and provides modification suggestions for the porting.
    • Structure byte alignment check: The tool checks the byte alignment of the structure variables in the source code.
    • Cache line alignment check: This function checks the 128-byte alignment of structure variables in the C/C++ source code to improve memory access performance.
    • Memory consistency check is to analyze and rectify the memory consistency issues in the user software.

This Help provides the operation guide and precautions for using the Kunpeng Porting Advisor.
