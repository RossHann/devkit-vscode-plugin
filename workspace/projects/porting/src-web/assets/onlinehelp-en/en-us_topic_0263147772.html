  • Uploading Files
    If the size of the file to be uploaded is greater than 1 GB or the size of the file after decompression is greater than half of the remaining drive space, you need to free up some drive space or manually upload the file to the server. In other cases, you can use the upload function on the WebUI.
  • Configuring a Proxy
    When you perform software package rebuilding or dedicated software porting, the server needs to access the Internet to obtain resource files. In an isolated network environment, you need to configure a proxy for accessing the Internet. If network access is available in the current environment, skip this section.
