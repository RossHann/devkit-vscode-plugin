Restoring a Software Porting Template

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

Procedure

  1. On the WebUI, click in the upper right corner and choose Software Porting Template.
  2. Click Restore. The page shown in Figure 1 is displayed.

    Figure 1 Restoring a software porting template

  3. Enter the password of the administrator and click OK.

    The restoration progress is displayed. After the restoration is complete, check the software porting template file in the /opt/portadv/resource/migration directory.

    • Do not close the current page during the restoration process.
    • If the restoration fails, rectify the fault based on the error message displayed or by referring to the FAQs in the user guide. For details about the failure information, see /opt/portadv/logs/porting.log.
    • /opt/portadv is the tool installation directory. Replace it with the actual directory.
