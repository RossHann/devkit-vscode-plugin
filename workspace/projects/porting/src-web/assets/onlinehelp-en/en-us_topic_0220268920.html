Backing Up a Software Porting Template

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

Procedure

  1. On the WebUI, click in the upper right corner and choose Software Porting Template.
  2. Click Back Up. The page shown in Figure 1 is displayed.

    Figure 1 Back up the software porting template

  3. Enter the password of the administrator and click OK.

    The backup progress is displayed. After the backup is complete, the backup file is saved in the /opt/portadv/portadmin/migration/backup directory by default.

    • Do not close the current page during the backup process.
    • If the backup fails, rectify the fault based on the error message displayed or by referring to the FAQs in the user guide. For details about the failure information, see /opt/portadv/logs/porting.log.
    • /opt/portadv is the tool installation directory. Replace it with the actual directory.
