Weak Password Dictionary Management

A weak password can be easily guessed or cracked. When a password is set or changed, the system checks the new password against the weak password dictionary. If a match is found, the user will be forced to reset the password for security purposes.

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

  • Users can view and search for weak passwords.
  • Only the administrator portadmin can add and delete weak passwords.
  • By default, 52 weak passwords are provided.
  • A maximum of 1000 weak passwords are allowed.

Procedure

  1. On the WebUI, click in the upper right corner and choose Weak Password Dictionary.
  2. Click Add.

    Figure 1 Adding weak passwords

  3. Enter the weak password to be added in the text box. Weak passwords must meet the following requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of the following characters:
      • Uppercase letters A to Z
      • Lowercase letters a to z
      • Digits 0 to 9
      • Special characters `~!@#$%^&*()-_=+\|[{}];:'",<.>/?

  4. Click OK.

    To delete a weak password, locate the weak password and click Delete in the Operation column. In the dialog box displayed, click OK.
