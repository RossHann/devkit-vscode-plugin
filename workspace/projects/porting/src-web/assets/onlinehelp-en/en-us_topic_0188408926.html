Logging In to the Kunpeng Porting Advisor WebUI

The following describes how to log in to the Kunpeng Porting Advisor web user interface (WebUI) by using Chrome 72.0 on Windows 10.

Browser Requirements

Table 1 lists the browser versions supported by the Kunpeng Porting Advisor.

Table 1 Browser requirements

Browser

Version

Chrome

Chrome 72.0 or later

Internet Explorer

Internet Explorer 11.0 or later

Edge

Microsoft Edge 79.0 or later

Internet Explorer 11.0 has compatibility issues with the compiler of the tool, when you click Undo to negate single-step code modification or when you apply batch code modification. You are advised to use the Google Chrome browser when performing these operations.

Procedure

  • By default, the system locks a user account upon five consecutive login failures. The user account will be automatically unlocked three minutes later.
  • The default system timeout period is 30 minutes. If no operation is performed on the WebUI within 30 minutes, the user will be automatically logged out of the WebUI. On the home page of Kunpeng Porting Advisor, the administrator can click and choose System Settings to set the session timeout period. The value ranges from 10 to 240 minutes.
  • For security purposes, change the initial password after the first login and change your password periodically.
  • By default, the tool supports concurrent access from a maximum of 10 common users. The number of administrators is not limited. On the home page of Kunpeng Porting Advisor, the administrator can click , choose System Settings, and set Maximum Number of Online Common Users. The value ranges from 1 to 20.
  • Only one active session is allowed for a user. If a user has logged in to the tool, re-login will terminate the previous session established.
  • The disclaimer message will be displayed upon your first login. Read it carefully and select the check box to agree with the content, click OK, and then you can use the tool. If you click Cancel, the tool exits.
  1. Open the browser on the local PC, enter https://Server IP address:Port number (for example, https://10.254.206.190:8084) in the address box, and press Enter.

    The default HTTPS port number is 8084. Before using this tool, ensure that port 8084 have been enabled on the operating system firewall.

  2. On the page displayed as shown in Figure 1, click Continue to this website (not recommended).

    If a website security alert is displayed, you can ignore this message or download the Huawei root certificate to shield this alert. For details, see Importing the Root Certificate for Kunpeng Porting Advisor.

    Figure 1 Security alert

    The login page is displayed, as shown in Figure 2. Table 2 describes the parameters.

    Figure 2 First login to Kunpeng Porting Advisor
    Table 2 Parameters on the Kunpeng Porting Advisor first login page

    Parameter

    Description

    User Name

    User name for logging in to the Kunpeng Porting Advisor.

    NOTE:

    The default administrator is portadmin. Upon the first login, you need to set the administrator password.

    Password

    Password for login. The password must meet the following complexity requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of uppercase letters, lowercase letters, digits, and special characters (`~!@#$%^&*()-_=+\|[{}];:'",<.>/?).
    • Cannot contain any space.
    • Cannot be the same as the user name.
    • Cannot be a password in the weak password dictionary.

    Confirm Password

    Password re-entered for confirmation.

  3. Enter the password of the administrator and click OK.

    The page shown in Figure 3 is displayed. Table 3 describes the parameters.

    Figure 3 Logging in to the Kunpeng Porting Advisor WebUI (not the first login)
    Table 3 Parameters on the Kunpeng Porting Advisor login page (not the first login)

    Parameter

    Description

    User Name

    User name for logging in to the Kunpeng Porting Advisor.

    NOTE:

    The default administrator is portadmin.

    Password

    Password for login. Change this password periodically for security purposes.

  4. Enter the user name and password, and click Log In.

    The common user is requested to change the password upon the first login. Change the original password as prompted. The password must meet the following complexity requirements:

    • Contain 8 to 32 characters.
    • Contain at least two types of the following characters:
      • Uppercase letters: A to Z
      • Lowercase letters: a to z
      • Digits: 0 to 9
      • Special characters `~!@#$%^&*()-_=+\|[{}];:'",<.>/?
    • Cannot be the same as the user name.
    • Cannot be the same as the old password.
    • Cannot be the old password in reverse order.
    • Cannot be a password in the weak password dictionary.

    The password of the Kunpeng Porting Advisor user is valid for 90 days. Change the password before the password expires. If the password has expired, you need to change the password after login.

    If the login is successful, the home page is displayed and the user name is displayed in the upper right corner of the page.
