Cache Line Alignment Check

Cache line alignment check: This function checks the 128-byte alignment of structure variables in the C/C++ source code to improve memory access performance.

Prerequisites

You have logged in to the Kunpeng Porting Advisor.

/opt/portadv is the default installation directory of the tool. The following uses this directory as an example. Replace it with the actual directory.

Procedure

  1. Choose Enhanced Functions from the navigation pane on the left.

    Select Cacheline Alignment as the check type, as shown in Figure 1.

    Figure 1 Cache line alignment check

  2. Set Source Code File Path. You can use either of the following methods to upload the file:

    • Click Upload to upload the package or folder. The package will be automatically decompressed during the upload process.
      • Only one compressed package in the following formats can be uploaded at a time: tar, tar.bz, tar.bz2, tar.gz, tar.xz, tbz, tbz2, tgz, txz, and zip The size of the source code package should be less than or equal to 1 GB. The decompressed file size should be less than or equal to half of the remaining drive space.
      • Only one folder can be uploaded at a time. The size of the folder must be less than or equal to half of the remaining drive space.
      • Internet Explorer does not support upload of folders. Use other browsers, such as Google Chrome and Microsoft Edge.
    • Manually upload the source code file to the specified path (for example, /opt/portadv/portadmin/cachecheck/) on the server. Grant the read, write, and execute permissions to the porting user. Then, click the text box to select the source code file path from the drop-down list. You can also manually enter the source code file path.

  3. Click Check. After the check is complete, click View Report. The page shown in Figure 2 is displayed.

    When editing the original code, save the modification in time. Otherwise, the modification may be lost due to session timeout.

    Figure 2 Cache line check report
    Table 1 Parameters in the cache line check report

    Parameter

    Description

    File List

    Lists the source code files analyzed.

    Original Source Code

    Displays the original source code. You can also move the cursor to the code with a red wavy line and click Peek Problem in the floating message box to view the source code suggestions. You can modify the source code. After the modification, click Save, Undo, or Cancel in the upper right corner as required.
    • The tool supports concurrent running of multiple cache line alignment check tasks.
    • To cancel a task, click Close during the analysis process.
    • If the source code does not need to be modified, a dialog box is displayed indicating that the operation is successful. The check report page is not displayed.
    • To download a source code file, select the file on the left and click the download icon.
