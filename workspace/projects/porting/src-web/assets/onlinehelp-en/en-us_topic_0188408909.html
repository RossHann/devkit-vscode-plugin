Home page

Figure 1 shows the home page of the Kunpeng Porting Advisor. Table 1 describes the home page.

Figure 1 Kunpeng Porting Advisor home page
Table 1 Parameters on the Kunpeng Porting Advisor home page

Area

Name

Description

1

Software Porting Assessment

Provides an entry for software porting assessment. You can create an analysis task for assessment in this area.

2

Historical Reports area

Displays reports generated.

3

Source Code Porting

Provides an entry for source code porting. You can create an analysis task for source code porting in this area.

4

Software Package Rebuilding

Provides an entry for software package rebuilding. You can analyze software packages and rebuild software packages applicable to the Kunpeng platform.

5

Dedicated Software Porting

Provides solution-specific software porting methods accumulated by Huawei expertise.

6

Enhanced Functions

The following enhanced functions are provided:

  • 64-bit mode check
  • Structure byte alignment check
  • Cache line alignment check
  • Memory consistency check
    NOTE:

    Enhanced functions: Before porting applications from 32-bit platforms to 64-bit platforms, you can use this tool to check the 64-bit mode running, structure byte alignment, cacheline alignment, and memory consistency issues that may occur when running on the Kunpeng platform. GCC 4.8.5 to GCC 9.3 on x86 and Kunpeng platforms are supported.

7

Current user

Displays the current login user and provides an entry for resetting the login password and logging out.

8

Settings

Provides an entry for user management, weak password dictionary, system configuration, log, dependency dictionary, software porting template, scanning parameter configuration, threshold setting, web server certificate, and certificate revocation list (CRL).

9

More

Provides an entry for:

  • Switching the language between simplified Chinese and English
  • Providing feedback
  • Viewing the online help, disclaimer, and Kunpeng Porting Advisor release information

10

Workspace/Drive space alarm area

Displays the total and remaining workspace or drive space information. A yellow alarm is displayed when the remaining space is less than 20% of the total space. A red alarm is displayed when the remaining space is less than 5% of the total space. The alarm is cleared when the remaining space is larger than 20% of the total space.

Red: indicates a critical alarm. If the remaining space is less than 1 GB, the source code porting, dedicated software porting, and software package rebuilding functions are unavailable. In this case, delete historical analysis records or release drive space.

Yellow: indicates a warning. In this case, the workspace or drive space is consumed fast and may affect the use of the tool. To prevent negative impact on the tool, delete historical analysis records or release drive space.

NOTE:

The user workspace is under the tool installation directory. At least 100 GB is required for installing Kunpeng Porting Advisor.

12

Case links

Provides links to typical cases and sample code for each function.

The tool supports concurrent running of tasks. You can run multiple tasks, including software porting assessment, source code porting, dedicated software porting, software package rebuilding, and enhanced functions, at the same time.
