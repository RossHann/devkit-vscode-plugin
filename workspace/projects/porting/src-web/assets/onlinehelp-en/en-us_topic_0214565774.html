Configuring Scan Parameters

Customize the baseline workload for estimating the code porting workload.

Prerequisites

You have logged in to the Kunpeng Porting Advisor WebUI.

Procedure

  1. Click in the upper right corner and choose Configure Scan Parameters.

    The Configure Scan Parameters page is displayed, as shown in Figure 1. Table 1 describes the parameters.

    Figure 1 Configure Scan Parameters
    Table 1 Parameters on the Configure Scan Parameters page

    Parameter

    Description

    Keyword Scan

    Specifies whether to continue the scan when keyword ARM, ARM64, or AArch64 is detected.

    Default value: Yes

    C/C++/Fortran/Go Porting Workload Estimation (lines/person month)

    Baseline workload for estimating the workload of C/C++, Fortran, or Go code porting.

    Value range: 1 to 99999

    Default value: 500

    Assembly Porting Workload Estimation (lines/person month)

    Baseline workload for estimating the workload of assembly code porting.

    Value range: 1 to 99999

    Default value: 250

    Display Workload Evaluation Results

    Specifies whether to display the estimated porting workload on the analysis report details page.

    Default value: No

    User Password

    Password of the current user.

  2. Click Modify, set parameters according to Table 1, enter the user password, and click OK.
