Downloading an Analysis Report

Prerequisites

The analysis task is complete.

Downloading an Analysis Report in the Historical Reports Area

  1. Choose Software Porting Assessment from the navigation pane on the left.
  2. In the Historical Reports area, locate the report to be downloaded and click , as shown in Figure 1.

    Figure 1 Historical Reports

  3. Click Download Report (.csv) or Download Report (.html).

Downloading an Analysis Report on the Analysis Result Page

  1. Choose Software Porting Assessment from the navigation pane on the left.
  2. In the list of historical reports, click the name of the report.

    The report page is displayed.

    Click View Report to open the report page.

  3. Click Download Report (.csv) or Download Report (.html) to download the analysis report.
